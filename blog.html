Entendendo o Outline do HTML

Entenda porquê não usar tantos H1 e como funciona o fluxo de outline do HTML.

04 de Junho de 2017    |    Fellipe de Palma

O Outline do documento HTML é a estrutura do documento que é gerada pelos títulos, títulos de formulários, títulos das tabelas e qualquer outro elemento que marque alguma seção de conteúdo no documento. Essas marcações de Outline podem ser usadas para várias iniciativas, por exemplo, um browser ou alguma outra interface possa criar uma Tabela de Conteúdo. Outras tecnologias assistivas podem usar também estas seções para formar a navegação, como em um leitor de tela, mas ou menos como acontece no WAI-ARIA, mas de maneira mais natural.

Os elementos de cabeçalho h1-h6, contribuem para o outline do documento HTML 5, criando uma estrutura hierárquica e definindo as seções da página. Os cabeçalhos são importantes para otimização de buscas e também para navegação em leitores de telas, já que usuários desses aparelhos navegam pelos títulos na página.

Browsers comuns não mostram o outline do documento portanto para visualiza-los é necessário usar uma ferramenta chamada HTML5 outliner(https://gsnedders.html5.org/outliner/). Mesmo que o nível dos cabeçalhos dos exemplos sejam diferentes, a saída como vimos é a mesma.

Pois bem, agora com o HTML5 temos novas formas de estruturar o documento. Temos os elementos de sectioning content — article, aside, nav e sections — que demarcam diferentes seções num documento e define o escopo para os elementos de cabeçalho (assim como o header e o footer).

Como fazer o efeito de dupla exposição no Photoshop

04 de Junho de 2017    |    Fellipe de Palma

01. Neste tutorial usaremos apenas duas imagens, a imagem da modelo e a imagem que fará parte da dupla exposição.

02. Pressione Ctrl+O e abra a foto da modelo, pressione a letra P do teclado para selecionar a Pen Tool, certifique-se de que a opção Paths esteja habilitada e crie um path ao redor da modelo.

03. Crie o Path ao redor da modelo, áreas como o cabelo serão trabalhadas posteriormente.

04. Ao terminar o Path clique com o botão direito do mouse dentro do Path criado e escolha a opção Make Selection, defina os valores abaixo e pressione Ok.

05. Vá em Select> Refine Edge e defina os valores abaixo.

06. Pressione Ok, sua seleção deverá ficar como a da foto abaixo, pressione Ctrl+J para duplicar a área selecionada para uma nova layer, pressione Ctrl+Shift+Alt+N para criar uma nova layer, pressione a letra D do teclado para resetar as cores de Foreground e Background para preto e branco, pressiona a letra X do teclado para alternar as cores de Foreground e Background, sua cor de Foreground deve ser a branca, pressione Alt+Backspace (a tecla acima do Enter) para preencher a layer com branco, deixe a layer abaixo da layer copiada.

07. Com a tecla Ctrl pressionada clique sobre a miniatura da layer copiada para ativar a seleção, vá em File> Place e adicione a segunda imagem, ajuste seu tamanho e largura, pressione Enter para finalizar a transformação.

08. Vá em Layer> Layer Mask> Reveal All para adicionar uma máscara na layer, clique sobre o pequeno ícone de corrente que existe entre a miniatura da imagem e a miniatura da máscara para retirar o link entre as duas coisas.

09. Clique na layer da imagem copiada, pressione Ctrl+J para duplicar a layer, deixe a layer acima da layer de textura, pressione Ctrl+Shift+U para retirar a saturação da imagem, pressione Ctrl+L para abrir a caixa de Levels e defina os valores abaixo.

10. Mude o modo de blend da layer para Screen.

11. Vá em Layer> Layer Mask> Reveal All para adicionar uma máscara na layer, pressione a letra D do teclado para resetar as cores de Foreground e Background para preto e branco, pressione a letra B do teclado para selecionar a Brush Tool, escolha um brush de pontas suaves e tamanho médio, aplique no documento como na foto abaixo.

12. Escolha uma cor qualquer para o Background (caso não queira a cor branca), no caso deste tutorial usamos a mesma cor utilizada na textura aplicada.

13. Vá em Layer> Layer Style> Gradient Map, crie um gradiente com as cores #e2d9d1, #52463b e #0e1133, mude o modo de blend do gradiente para Color e aplique.
